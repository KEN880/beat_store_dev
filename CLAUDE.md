# KEN&BAJ — beat presentation site

Личный сайт-витрина для битов продюсера **KEN&BAJ** (Trap/Drill, RnB/Soul).
Контакт: **kenmusic888@gmail.com**. Это НЕ маркетплейс в стиле BeatStars — это
атмосферная вертикальная «прогулка» (tour) сверху вниз, где каждая секция = один
бит со своим визуальным вайбом.

## Файлы

- `index.html` — **корень сайта = каталог** (копия `catalog.html`). При правках каталога: `cp catalog.html index.html`.
- `beat-site.html` — иммерсивный тур (старый лендинг). Живёт на `/tour.html` (копия: `cp beat-site.html tour.html`). Single-file: весь HTML/CSS/JS внутри.
- `catalog.html` — плейлист-каталог из Supabase: табы плейлистов, поиск, фильтры по жанру, сортировка, корзина, липкий плеер (реальное аудио из Storage, симуляция как fallback).
- `admin.html` — админка (Supabase Auth): CRUD битов (+загрузка mp3 в bucket `audio`, цвета обложки, порядок, публикация), плейлисты, ссылки линктри. Креды и project-id — в Obsidian: `OBSIDIAN/MAC/KENBAJ/KENBAJ - Текущий статус.md`.
- `links.html` — линктри (мобайл-фёрст), карточки из таблицы `links`, управляются из админки.
- `design/variants/` — 6 скин-мокапов (noir/y2k/lookbook/terminal/zine/liquid) + `playlist.html` (статичный мокап каталога). Скин ещё не выбран.
- `ken-baj-logo.svg` / `ken-baj-logo.png` / `ken-baj-logo-black.png` — вордмарк «KEN&BAJ» (обведён в кривые из Liberation Sans Bold; точный Archivo — см. «Осталось»).

## Бэкенд (Supabase `kenbaj-beats`, eu-central-1)

- Таблицы: `beats`, `playlists`, `playlist_beats`, `links`. RLS: anon читает published, authenticated пишет всё. Storage bucket `audio` (public read).
- Ключи (URL + anon key) захардкожены в трёх страницах — это нормально, anon key публичный.
- Иконки: `@tabler/icons-webfont@2.47.0` с jsDelivr (cdnjs-путь 2.47.0/iconfont — 404, не использовать).
- Деплой: git push в main → Vercel (beat-store-dev.vercel.app).

## Архитектура сайта

Один файл, без сборки. Открывается локально в браузере.
- Шрифты: Google Fonts CDN — Anton, Archivo, DM Serif Display (italic), Space Mono.
- Иконки: Tabler Icons CDN (cdnjs).
- Секции на весь экран (`.z`, min-height:100vh), вертикальный скролл.

Структура секций (сверху вниз):
- `#z0` — интро, тёмный минимал, большое имя KEN&BAJ (Anton) с полосами-stripes.
- `#z1` `.z-trap` — бит **midnight**, trap/dark, красный slash, «night» курсивом DM Serif Display.
- `#z2` `.z-emo` — бит **luv**, мир в стиле old Lil Uzi Vert, градиент-закат, дудлы (SVG), глитч-имя cyan/magenta, японская вертикаль «ビート · 音».
- `#z3` `.z-rage` — бит **ash**, rage/Travis, техническая сетка (SVG), тэг «utopia // 002—rage».
- `#z4` `.z-mel` — бит **saint**, melancholy/Juice WRLD, фиолетовый, большое «999», иконка глобуса.
- `#out` `.z-out` — контакты, mailto:kenmusic888@gmail.com, соцсети.
- `footer` — баркод (SVG), «cat. no. 001», «KEN&BAJ © 2026».
- slim-плеер прибит к низу.

## Дизайн / палитра

- База: `--ink #0b0b0a`, `--paper #f5f2ec`.
- Акцент на зону (CSS-переменная `--acc`, меняется по скроллу): trap `#b5121b`, emo/Uzi `#ff4fa3` (+cyan `#46e0e0`), rage `#9c6b3d`, melancholy `#8a5cff`.
- **Швы между зонами сведены в непрерывный градиент**: цвет низа одной зоны = цвет верха следующей. Поток: тёмное имя → midnight → закат Uzi (`#e8843c`) → ash (оранжевый сгорает в темень) → фиолетовый saint → тёмный контакт. Это ключевая фишка, которую важно не сломать при правках фонов.
- Зерно (grain) поверх секций через `.grain::after` (SVG-шум).

## JS (внутри `<script>` в конце файла)

- `beats[]` + `acc[]` — данные для плеера.
- IntersectionObserver `zo` (threshold .55): активная зона → подсветка точки в боковой навигации + смена `--acc` + текст плеера через `setBeat(i)`.
- IntersectionObserver `io` (threshold .3): scroll-reveal анимации (`.reveal` → `.in`).
- Плеер сейчас **ненастоящий**: `setInterval` симулирует прогресс, реального аудио нет.
- Кастомный курсор (точка + догоняющее кольцо, lerp), скрыт на тач-устройствах.
- Навигация прячется при скролле вниз. Слева — **рейл-карта тура** (`#rail`): номер + имя зоны (`data-name`), активный пункт подсвечивается из `zo`-обсервера (заменил прежние точки `#dots`).
- **Эквалайзеры**: hero-бары в `#z0` (`.eqhero`, ambient) и мини-EQ в плеере (`.player .eq`, анимируется при `body.isplaying`). Активная `.pb` играющего бита пульсирует (`.pb.on`) и переключает иконку play/pause.
- **Арт по зонам** (всё чистый CSS/SVG, без ассетов): midnight — луна + горизонт + скайлайн + 2-й слэш; luv — больше дудлов (сердца, звёзды); ash — летящие угли (`#embers`) + перспектива сетки; saint — гало-кольцо + звездопад (`#stars`).

Проверка JS после правок: вытащить содержимое `<script>` и прогнать `node --check`.

## Состояние

Готово: вся структура тура, идентичность (KEN&BAJ / kenmusic888@gmail.com),
плавные градиентные переходы между всеми зонами, лого-ассеты, Sequoia-зум фона

**Уникальный арт-дирекшн на каждый бит** (концепция «альбом = вселенная», см.
`design/concept-hiphop.md`): midnight = AWGE-брутализм (лево, штамп TESTING,
redacted-кредит, скайлайн); luv = IGOR пастель-глитч (центр, хромо-глитч, дудлы);
ash = Travis-дистопия (право, скос, варнинг-лента, координаты, угли); saint =
Chromakopia-нуар (центр, свечение, гало/999, леттербокс, звездопад).
